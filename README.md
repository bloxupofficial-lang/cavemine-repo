# Mine A Cave | Roblox Game
Repository for Roblox front-page <ins>**Mine A Cave**</ins>.

# What is it about?
<ins>Incremental Simulator</ins>

* Purchase items
* Wait for items to produce
* Buy more items with production of items
* Get more items

Mine in Cave for items, sell for money, and use that money to get better items.

Minecarts can be used for faster mobility to get to the surface.

The deeper you go, the better ores there are, but there are pursuers to stop you.

There are over ***20*** different kinds of ores you can get!
